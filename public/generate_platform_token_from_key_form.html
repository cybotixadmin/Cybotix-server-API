The public keys needed here can made in a range of ways. Below are commands that will make the required keys using openssl.
openssl genpkey -algorithm RSA -out company.private_key.pem -pkeyopt rsa_keygen_bits:2048
openssl rsa -pubout -in company.private_key.pem -out company.public_key.pem
